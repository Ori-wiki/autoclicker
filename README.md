# Simple AutoClicker (Chrome Extension)

Автокликер-расширение для Chrome/Chromium с профилями, макросами, условиями остановки и безопасным запуском.

## Что добавлено

- Профили настроек (несколько сценариев, автосохранение активного профиля)
- Горячие клавиши `start / pause / stop` в профиле + глобальные команды расширения
- Случайный разброс интервала (`Jitter %`)
- Лимиты по числу кликов и времени
- Клик по шаблону точек (`x,y` по очереди)
- Условия остановки:
  - по цвету в точке (цвет + tolerance)
  - по CSS-селектору/маркеру
  - при смене окна/вкладки
  - при изменении URL (привязка к текущей вкладке)
- Привязка к конкретной вкладке/URL
- Запись и воспроизведение макросов (клики + клавиши + задержки)
- Логи и статистика (сессия + lifetime)
- Защита:
  - emergency-stop по `Esc`
  - блокировка кликов вне безопасной зоны
- Мини-оверлей на странице: статус, профиль, таймер, CPS, счетчик
- Импорт/экспорт конфигов (`JSON`)
- Планировщик старта: сразу / через N секунд / в заданное время
- Headless/CLI-подобный запуск через поле аргументов в popup
- Self-test перед стартом (валидность профиля, зона, селектор, фокус)

## Горячие клавиши

### Внутри страницы (из профиля)

- `Start hotkey`
- `Pause hotkey`
- `Stop hotkey`
- `Esc` - экстренная остановка

### Глобальные команды расширения

- `Ctrl+Shift+1` - Start
- `Ctrl+Shift+2` - Pause/Resume
- `Ctrl+Shift+3` - Stop

Изменить можно в `chrome://extensions/shortcuts`.

## Установка

1. Открой `chrome://extensions`.
2. Включи **Developer mode**.
3. Нажми **Load unpacked** и выбери папку проекта.
4. После изменений в коде жми **Reload** у расширения.

## Использование

1. Открой нужный сайт (не `chrome://...`).
2. Открой popup расширения.
3. Выбери/создай профиль и настрой параметры.
4. Нажми **Save Profile**.
5. Запусти **Start** (или hotkey).
6. Пауза - **Pause/Resume**, остановка - **Stop** или `Esc`.

## Headless/CLI аргументы (в popup)

Пример:

```text
--cps 12 --jitter 10 --duration 30 --clicks 500 --left --double --start-now
```

Поддержка:

- `--cps <number>`
- `--jitter <number>`
- `--duration <sec>`
- `--clicks <count>`
- `--left | --middle | --right`
- `--single | --double`
- `--start-now`
- `--delay <sec>`

## Структура

- `manifest.json` - манифест MV3 + команды
- `background.js` - обработка global hotkeys (commands)
- `popup.html` / `popup.css` / `popup.js` - UI и управление профилями
- `content.js` - движок кликов, макросы, stop conditions, overlay

## Ограничения

- Контент-скрипт не работает на `chrome://...` страницах.
- Проверка цвета основана на `background-color` элемента в точке (не скриншотный пиксель).
- Симуляция клавиш макроса зависит от ограничений конкретного сайта.
