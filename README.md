# Simple AutoClicker (Chrome Extension)

Простое расширение-автокликер для Google Chrome.

## Возможности

- Клик по текущей позиции курсора
- Настройка частоты кликов (`Clicks/sec`)
- Отдельные горячие клавиши для запуска и остановки
- Выбор кнопки мыши (`Left / Middle / Right`)
- Выбор типа клика (`Single / Double`)
- `Jitter (%)` для случайного разброса интервала
- `Esc` для экстренной остановки

## Требования

- Google Chrome (или Chromium-совместимый браузер)

## Установка и запуск

1. Открой Chrome и перейди на страницу `chrome://extensions`.
2. Включи переключатель **Режим разработчика** (Developer mode) в правом верхнем углу.
3. Нажми **Load unpacked** (Загрузить распакованное расширение).
4. Выбери папку проекта:
   `c:\Users\kozle\ww\autoclicker`
5. Расширение появится в списке. Если нужно, закрепи его через иконку пазла в панели браузера.

## Как пользоваться

1. Открой нужный сайт (обычная вкладка, не `chrome://...`).
2. Нажми на иконку расширения.
3. Настрой:
   - `Clicks/sec`
   - `Jitter (%)`
   - `Mouse button`
   - `Click type`
   - `Start hotkey` и `Stop hotkey`
4. Нажми **Save settings**.
5. Наведи курсор на нужный элемент.
6. Запусти автоклик кнопкой **Start** или горячей клавишей `Start hotkey`.
7. Останови кнопкой **Stop**, `Stop hotkey` или клавишей `Esc`.

## Важно

- На страницах `chrome://...` контент-скрипты не работают (ограничение Chrome).
- После изменения кода расширения нажми **Reload** на `chrome://extensions`.
- Если автоклик не работает, обнови вкладку сайта после перезагрузки расширения.

## Структура проекта

- `manifest.json` - манифест расширения (MV3)
- `popup.html` - интерфейс popup
- `popup.css` - стили popup
- `popup.js` - логика popup и сохранение настроек
- `content.js` - логика автоклика на странице
